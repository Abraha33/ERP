# ERP Satelite Operativa

Sistema empresarial completo construido por un solo founder en 14 meses.

## Estrategia de construccion

FASE 1 - App Satelite    movil, importa Excel del SAE, operaciones de campo
FASE 2 - ERP Basico      catalogos, compras, ventas, inventario
FASE 3 - ERP Completo    contabilidad, RRHH, reportes avanzados
FASE 4 - CRM             modulo comercial sobre el ERP
FASE 5 - Offline-First   resiliencia sin internet

## Fase 1 - App Satelite MVP (Mes 1-3)
App movil para trabajo de campo. Se alimenta de Excel exportados del SAE.
Opera en paralelo al SAE sin reemplazarlo todavia.
- Importacion de catalogo desde Excel
- Roles: administrador, encargado, empleado
- Recepcion, traslados, misiones de conteo, arqueo de caja

## Fase 2 - ERP Basico (Mes 4-7)
Modulos operativos esenciales. Empieza a reemplazar el SAE.
- Catalogos, Compras, Ventas, Inventario

## Fase 3 - ERP Completo (Mes 8-10)
Completa el reemplazo del SAE.
- Contabilidad, RRHH, Reportes gerenciales, Auditoria

## Fase 4 - CRM (Mes 11-12)
- Comunicacion interna, transcripcion de audio, casos, pipeline de ventas

## Fase 5 - Offline-First (Mes 13-14)
- Base de datos local, sync automatico, resolucion de conflictos
