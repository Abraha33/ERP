# Scripts SAE (import / export)

Coloca aquí (o en módulos adyacentes) los CLI Python que lean **CSV/XLS** del SAE y escriban en Supabase (PostgREST o `DATABASE_URL`), y el export inverso.

- Variables: ver `.env.example` (`SAE_EXPORT_PATH`, `SAE_IMPORT_PATH`, `SUPABASE_*`, `DATABASE_URL`).
- No commitear datos reales ni `.env`.
