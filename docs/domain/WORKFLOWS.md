# WORKFLOWS

## Objetivo

Documentar los flujos principales del negocio para que arquitectura, UI y automatizaciones hablen el mismo idioma.

## Flujo comercial

1. Lead
2. Oportunidad
3. Cotización
4. Pedido
5. Factura
6. Recaudo
7. Seguimiento postventa

## Flujo de compras

1. Solicitud
2. Aprobación
3. Orden de compra
4. Recepción
5. Factura proveedor
6. Causación/pago

## Flujo de inventario

1. Entrada
2. Ubicación
3. Movimiento interno
4. Salida
5. Ajuste
6. Conteo

## Flujo de cartera

1. Emisión factura
2. Vencimiento
3. Recordatorio
4. Gestión de cobro
5. Recaudo
6. Conciliación

## Reglas

- Todo flujo debe dejar trazabilidad
- Todo cambio de estado relevante debe ser auditable
- Los flujos aprobatorios deben tener actor y timestamp
